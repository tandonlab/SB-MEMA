# SB-MEMA
Surface-Based Mixed-Effects Multilevel Analysis for both subdural grid and stereotactic depth electrodes.

# Requirements
- AFNI/SUMA
- Freesurfer
- R
- MATLAB

Install instructions assume OSX architecture with /bin/tcsh terminal.
